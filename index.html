Job Search Managment System

This project was created using Rails and I used Bootstrap for the styling aspects, and uploaded using Heroku. As we were finishing up our classes at Star City Code School and began the job search I was trying to come up with an idea for another project that would incorporate both rails and bootstrap. I figured why not make a web app that would be mobile based that I could use to keep track of all the jobs I was applying for? I made this with the purpose of being able to access the app from my iphone when I was not around my computer or wifi t quickly be able to see jobs I have applied for, add a new job posting to database, as well as upload cover letters that were specific for each job and have easy access to them. I had an option to create new users, although currently all users can see all jobs regardless of user. I also set it up so you needed to be a registered and I made this authentication by using sorcery. Overall it is a basic web app that has been very useful for both keeping track of places I have applied but also to quickly add potential new opportunities I come accross when I am not around my computer.

Soccer League Managment Sysytem

This is the project for weeks 9-10 of the Star City Code School that a class mate and I created. The goal of this project was to create a web app that was a hypothetical soccer league managment system. You needed to be able to create players, create teams, and assign those players to specific teams. We also created a user logging in and out feature using Sorcery. Another feature that was added was a "Contact Us" feature that would allow someone to enter their name and email and an information request would be sent to sccssoccerproject@gmail.com as a way for a user to ask for more information about the league. We are in the process of adding in a feature that will alow a user to enter in their email and using the ActionMailer feature of rails it will generate an email that will go directly to the users entered email. Styling was done on a limited basis as this project focused more on using rails generators, Sorcery, and ActionMailer. Another skill that was important from this project was this was our first one using collaboration since in the professional world that will be the majority of our work. When registering any email will work but will not be confirmed for valid address and no confirmation will be sent. We created it simply for a functional purpose. You can then use that email and password to log in on the login screen. This will not create a new "player", that still needs to be done on the create players page.
